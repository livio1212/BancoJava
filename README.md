# 🏦 Banco Digital em Java

Bem-vindo(a) ao repositório do **Banco Digital**, um projeto desenvolvido em Java para simular operações bancárias de forma simples e eficiente. Este projeto foi criado com o objetivo de aprimorar habilidades no desenvolvimento backend e explorar conceitos fundamentais da programação orientada a objetos.

---

## ✨ **Funcionalidades**

- **Criação de contas bancárias**: Conta Corrente e Conta Poupança.  
- **Operações bancárias**:  
  - Depósitos  
  - Saques  
  - Transferências entre contas
  - consultar conta
  - Cadastrar
- **Consulta de saldo e extrato** detalhado para cada tipo de conta.
